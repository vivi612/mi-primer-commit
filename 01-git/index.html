Mi primer Commit en Git

  1. Crear un carpeta vacia
  2. En VS code, abrir la carpeta creada
  3. Usar el menú de terminal, y seleccionar la opción New Terminal
  4. En la terminal, escribir el comando git init
  5. Se crea una subcarpeta 01-git y dentro un archivo index.html
  6. Verifico el estado actual de git, con el comando git status
